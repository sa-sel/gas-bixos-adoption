O que é cada página da planilha?
  1. "Buscar" - serve apenas para procurar os dados de um bixo e de seu respectivo padrinho/madrinha. Você vai colocar o nome do bixo na célula "Nome" (D6) e os dados vão aparecer automaticamente.
    • Isso é útil para quem estiver moderando o grupo do Facebook - ao aprovar a entrada de um bixo no grupo, costuma-se enviar para ele uma mensagem com as informações do veterano que o adotou e, nesse caso, o jeito mais fácil de obtê-las é através dessa busca.
    • Basta apertar o botão "ENTROU NO GRUPO" e na planilha será marcado que aquele bixou entrou no grupo do Facebook. Ou seja, não é necessário buscar por esse bixo na "Lista dos Meliantes" e manualmente marcar o "Entrou no grupo?" como verdadeiro.
  2. "Lista dos Meliantes" - é a planilha principal, em que você poderá ver todas as informações sobre os bixos cadastrados e já sorteados.
    • O "Situação (Elétrica)" diz respeito a se o bixo vai ficar na Elétrica CAASO.
  3. "Números" - serão mostrados números relacionados às quantidades dos diversos tipos de bixos veteranos. Não é necessário fazer nada, essa página será atualizada automaticamente.
  4. "Bixos Perdidos" - serão mostrados todos os bixos que ainda não entraram no grupo do Facebook. Não é necessário fazer nada, essa página será atualizada automaticamente.
  5. "DB - Bixos" - é aonde serão colocados os bixos que se deseja sortear para os veteranos. Por exemplo, quando sair uma chamada da Fuvest, você vai pegar os nomes e ênfases dos bixos, colocar lá e, no menu Scripts (lá em cima), apertar o botão "Sortear bixos".
  6. "DB - Veteranos" - é aonde você vai colocar os dados dos veteranos e vetearnos que se inscreveram na Adoção dos Bixos. É necessário incluir, pelo menos, nome, email e ênfase para que o script consiga rodar com sucesso. A coluna "Monobixo" diz respeito a se aquela pessoa optou por adotar apenas um bixo ou não.
  7. "Template do Email" - é aonde você vai colocar e visualizar o... bom, o template do email que será enviado para os padrinhos/madrinhas quando os bixos forem sorteados.
    • Você deve escrever o corpo do email e pode escrever o título do email.
    • Para ver um preview de como vai ficar o email, basta apertar no botão "VISUALIZAR EMAIL".
    • Se você não escrever um título pro email, será usado o título padrão: "Adoção de Bixos da Elétrica 20XX"".
    • Se você incluir a variável $(nomeVeteran@) no texto (tanto título quando corpo), quando os emails forem enviados ela será substituída pelo primeiro nome de cada veterano. Por exemplo, "Olá, $(nomeVeteran@)!" irá virar "Olá, Lucas!", no email enviado para o Lucas Viana.
    • É obrigatório incluir a variável $(listarBixos) no texto do corpo do email (ela não é válida no título). Quando os emails forem enviados ela será substituída pela lista dos bixos adotados por aquela pessoa. O formato é <li>NOME_DO_BIXO (ÊNFASE_DO_BIXO)</li>, para cada bixo e, portanto é necessário que essa variável esteja inclusa dentro de uma lista. Por exemplo <ol>$(listarBixos)</ol>, para uma lista ordenada (numerada), ou <ul>$(listarBixos)</ul> para uma lista não ordenada (bulletpoints).
    • O corpo do email deve ser escrito formatado em HTML. Caso você não tenha familiaridade com a linguagem, recomendo usar essa ferramenta.
    • Os emails só serão enviado se a conta rodando o script for a sasel@eesc.usp.br
    • Tenha em mente que, no final do texto, será inclusa a assinatura da SA-SEL:
      --
      SA-SEL - Secretaria Acadêmica da Engenharia Elétrica
      Departamento de Engenharia Elétrica e Computação
      Escola de Engenharia de São Carlos


O que faz cada comando do menu "Scripts"?
  • "Sortear bixos" - é o comando principal, o script vai:
    1. Pegar todos os bixos da "DB - Bixos" e sortear entre os veteranos da "DB - Veteranos".
    2. Incluir todos esses dados no final da "Lista dos Meliantes" e limpar a "DB - Bixos".
    3. Enviar os emails para cada um dos veteranos que receberam bixos com os dados destes.
    4. Pedir ao usuário o título da chamada que acabou de ser processada (ex: Primeira Chamada, Segunda Chamada)
    5. Criar uma página na planilha da chamada que acabou de ser processada, com os bixos que acabaram de ser sorteados, para ser printada e divulgada no grupo do Facebook, incentivando as pessoas a caçarem os bixos. Ao fazer isso, ele vai esconder (não excluir) todas as páginas das chamas anteriores.
  • "DB Bixos - toda" - vai limpar todos os dados da página "DB Bixos".
  • "DB Veteranos - toda" - vai limpar todos os dados da página "DB Veteranos".
  • "Lista dos Meliantes - linhas vazias" - vai excluir todas as linhas vazias da página "Lista dos Meliantes".
  • "Lista dos Meliantes - toda" - vai limpar todos os dados da página "Lista dos Meliantes".


Alguns outros pontos importantes:
  • Tenha muito cuidado ao editar a planilha, espcialmente células protegidas.
  • Não mude os nomes das páginas da planilha nem das células nomeadas, senão vai dar pau no script.
  • Nas páginas que já existem da planilha, não crie colunas à esquerda dentro das que já existem. É possível apenas criar colunas à direitas das que já existem, senão vai dar pau no script.
  • O ideal é não marcar a checkbox do "Entrou no grupo?" manualmente na "Lista dos Meliantes", mas sim através do botão na "Buscar".
  • Os emails para os padrinhos/madrinhas só serão enviado se a conta rodando o script for a sasel@eesc.usp.br
